Threat Model Name: Big Data Analyser

Owner: Jayesh Chandorkar

Reviewer:

Contributors: Jayesh Chandorkar, Abhay Thappan, Johnson Quadras

Description: The Big Data analyser portal provides the insights into the IDP session data. The portal has ability to analyse large data and provide analytics and predictions

Assumptions: 1 The deployment of meteor portal will be done in SunGard data center. 2 The Big Query is provided by Google and it is important to have internet access on the deployment server.

External Dependencies: The Big Query is being used to store the Big Data on which analytics to perform. The Big Query provides fastest access to the data and provide required infrastructure out of the box. Google Prediction API is being used to predict future actions depending on the usage of the IDP portal.
